# Exam System
## Installation

```bash
  git pull repository_url
```
```bash
  cd project_file
```
```bash
  rename .env.example to .env
```
```bash
  composer install
``` 
```bash
  php artisan migrate
```
```bash
  php artisan db:seed
```
```bash
  php artisan serve
```
For sending mail notifications start queue server
```bash
  php artisan queue:work
```
